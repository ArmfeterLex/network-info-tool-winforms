# Network Info Tool WinForms

Многофункциональная утилита на C# Windows Forms для проверки сетевых подключений и получения информации о сети.

## Функциональность

### 🌐 Работа с URL и URI
- Разбор URI через UriBuilder
- Извлечение хоста, порта, схемы
- Формирование полного URI

### 📡 DNS и IP-адреса
- Получение IP-адресов домена (yandex.ru)
- Определение IP по имени хоста

### 🏓 Ping
- Проверка доступности сайта
- Вывод времени ответа, TTL, размера буфера

### 🔌 Проверка подключения
- Определение наличия интернет-соединения через wininet.dll
- Проверка LAN/WAN подключения через sensapi.dll
- Проверка доступности URL

### 👤 Информация о системе и пользователе
- Имя компьютера (Environment.MachineName)
- Домен и имя пользователя
- Текущий пользователь Windows (WindowsIdentity)

## Интерфейс

- button1 — работа с UriBuilder
- button2 — DNS для yandex.ru
- button3 — имя компьютера
- button4 — ping для rusproject.narod.ru
- button5 — DNS по пустой строке
- button6 — проверка URL из textBox1
- button7 — проверка интернет-подключения
- button8 — домен и имя пользователя
- button9 — проверка LAN/WAN
- button10 — текущий пользователь
- listBox1 — вывод результатов ping
- textBox1 — ввод URL для проверки

## Куда можно встроить

- В диагностические утилиты для системных администраторов
- В программы мониторинга сети
- В обучающие материалы по работе с сетью в C#
- В состав более крупных сетевых инструментов

## Требования

- Windows
- .NET Framework / .NET Core с WinForms
- Доступ к интернету для некоторых функций

## Примечание

Программа использует как управляемый код (.NET), так и неуправляемые вызовы (DllImport) для wininet.dll и sensapi.dll. Функция CheckUrl проверяет доступность URL через HTTP-запрос.
